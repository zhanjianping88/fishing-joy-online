About Ocean Arcade Game

Ocean Arcade Game is a small web game project built around fishing runs, a few side modes, and pages that support the game without getting louder than it.

What this site is for

The goal is simple: open the page, find the game fast, and have a reason to come back later. That means playable screens come first, while support pages stay short and useful.

Open source foundation

The featured fishing game is adapted from the open source repository imtonyjaa/ggemu-fishing-joy. The original MIT license and attribution are preserved on the license page.
